\doxysubsection{Detailed Description}
Classe Filtre\+Salle\+Form\+Type pour créer un formulaire de filtrage pour les salles. Cette classe étend Abstract\+Type et définit la structure du formulaire pour filtrer les salles en fonction de divers critères tels que l\textquotesingle{}étage, l\textquotesingle{}orientation, la présence d\textquotesingle{}ordinateurs, et de systèmes d\textquotesingle{}acquisition.
