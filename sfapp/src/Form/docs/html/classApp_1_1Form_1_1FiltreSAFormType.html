Detailed Description

Classe FiltreSAFormType pour créer un formulaire de filtre pour les Systèmes d'Acquisition (SA). Cette classe étend AbstractType et définit la structure du formulaire pour filtrer les SA en fonction de différents critères tels que l'état et la localisation.
