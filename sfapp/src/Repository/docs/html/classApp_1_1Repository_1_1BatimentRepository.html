App\Repository\BatimentRepository::tableauBatimentsNomID ( )

Retourne un tableau associatif de bâtiments.

La méthode exécute une requête DQL pour récupérer les id et noms des bâtiments. Elle transforme ensuite les résultats en un tableau associatif où les clés sont les noms des bâtiments et les valeurs sont leurs identifiants.

Returns
array<string, int> Un tableau associatif où les clés sont les noms des bâtiments et les valeurs sont leurs identifiants.
